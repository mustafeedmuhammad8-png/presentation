Introduction (Concept & Vision)

Overview: NutriScan addresses the growing global concern regarding dietary health and obesity.

Core Technology: It leverages advanced Artificial Intelligence (AI) and Machine Learning (ML) to provide instantaneous nutritional analysis.

User Experience: Simplifies calorie tracking and macronutrient monitoring, making it accessible for individuals with varying levels of technical expertise.

AI Image Recognition Active

The AI Engine vs. Raw Chatbots

Raw AI Chatbots

  • Provide text-heavy, difficult to read responses.
  • No Persistence: Data disappears after the chat ends.
  • Lacks daily goal tracking or visual progress rings.

NutriScan System

  • Integrates Google Gemini's Vision capabilities seamlessly.
  • Persistent Dashboard: Locks data into a daily progress loop.
  • Transforms raw AI output into a structured health ecosystem.

Background and Motivation

The Health Crisis

Maintaining a balanced diet is a challenge due to the prevalence of processed foods and busy lifestyles.

The Friction of Existing Apps

Traditional calorie tracking apps often require tedious manual entry, leading to low user retention. Over 90% of users experience "logging fatigue".

Our Motivation

To eliminate the friction between eating a meal and logging it by utilizing a "1-Click" AI scanning approach.

Problem Statement

Manual Entry Limitations

Existing nutrition apps are often complex, requiring users to manually search for and log every food item. This causes massive user drop-off.

The Need for Automation

There is a need for a seamless, AI-powered solution that can identify food from images and provide accurate nutritional data without manual input.

Relevant Work / Literature Review

Ref. Title Summary Remarks
[1] MyFitnessPal / FatSecret Platforms with extensive databases for manual tracking. They lack the seamless "Image-to-Macro" conversion. High friction.
[2] Raw LLMs (Gemini/ChatGPT) Predictive AI analysis using computer vision capabilities. Fails to act as a tracking dashboard; lacks persistent goal UI.
[3] NutriScan (Our Proposal) 1-Click AI image scanning integrated into a dedicated tracking UI. Bridges the gap: High AI accuracy + Zero manual friction.

Aim and Objectives

Project Aim:

To develop an AI-first, low-friction application that completely automates macronutrient assessment and visualizes long-term dietary health.

To develop a mobile-first web application for automated food recognition.

To provide real-time estimates of calories, proteins, carbohydrates, and fats.

To implement a minimalist and intuitive UI/UX for better user engagement.

Methodology (System Architecture)

The NutriScan system follows a serverless architecture to ensure speed and cost-effectiveness.

Frontend UI (React & Tailwind)

Provides a clean, responsive user interface utilizing a minimalist design.

AI Brain (Gemini Vision API)

Analyzes images and extracts precise nutritional data instantly.

Hosting (Vercel / Netlify)

Ensures the application is live, extremely fast, and globally accessible.

Data Storage (Local Storage)

Saves user logs locally on the device to maintain privacy and speed.

Methodology (Data Flow)

1. Capture

User captures food image via UI.

2. Encode

Image converted to Base64.

3. API Prompt

Gemini API returns structured JSON.

4. Dashboard

Frontend parses JSON to UI.

Methodology (Handling Edge Cases)

The 2D Depth Limitation:

Computer vision cannot inherently measure volume (depth/ml) from a standard 2D image without reference points.

The "Human-in-the-Loop" Solution:

1. AI identifies the liquid type (e.g., Orange Juice) and base macros per 100ml.

2. The UI seamlessly prompts the user to select the container size to ensure 100% calculation accuracy.

Scope & Application

Scope: Focuses on standard local and international meals, acting entirely as a software platform without the need for external hardware sensors.

Real-World Applications:

Personal Health

Simplifying calorie tracking for individuals managing weight or obesity.

Student Planning

Dietary planning for busy university students and young professionals.

Fitness & Clinical

Tracking strict macros for athletes or diabetic carb-counting.
